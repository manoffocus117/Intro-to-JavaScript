## summary of this module 

1. what is javascript?
      => javascript is a high-level and interpreted programming language.
2. what can javascript do?
      => it can change or modify or update content dynamically, it can control medial,
       it can animate, and pretty much everything else.
3. what is variable? 
      => variable is a container used to store data.
4. what can a variable store? 
      => a variable can store string (words), number, boolean (true or false), etc.
5. what are the data types in javascript?
      => javascript data types are : string, number, boolean, null, undefined, symbol and bigint.
6. what are the naming conventions in javascript? 
      => camelCase, PascalCase and snake_case.
7. what are the rules to declaration a variable in javascript?
      => cannot be a reserved keyword 
      => should be a meaningful 
      => cannot start with a number
      => cannot contain space or hyphen
      => case sensitive
8. what are the types of number in javascript?
      => integer ( 50 ) and float ( 44. 9 )
9. what are the use of addition or concat ( + ) operator in javascript? 
      => when + is used with numbers, it is used to add the number.
      => when + is used with number and string, it is used to concatenate them.
10. what are the use of ( - , * , / , % and ** ) operator in javascript?
      => ( - ) or subtraction is used to subtract or reduce the value  
      => ( * ) or multiplication is used to multiply the value  
      => ( / ) or division is used to divide the value  
      => ( % ) or modulus is used to return the reminder of the value  
      => ( ** ) or exponentiation is used to power the value ( like 5 ^ 5 )  
11. what is NaN and Infinity in javascript?
      => when we try to perform some arithmetic or mathematical operation between two strings, it gives us NaN (not a number)
      => when we try to divide a number by 0 ( zero ), it gives us infinity.
12. what are the number methods in javascript? 
      => Number(), parseInt(), parsFloat(), toFixed(), toString(), valueOf(), isNaN(), etc.
13. what are the arithmetic operators in javascript?
      => addition, subtraction, multiplication, division, modulus, exponentiation, increment and decrement.
14. what are the assignment operators in javascript?
      => add and assign, subtract and assign, multiply and assign, divide and assign,
      modulus and assign, exponentiation and assign ( **= ), logical and ( && ) assign, 
      logical or ( || ) assign, nullish coalescing assign ( ??= ) and spread ( ... ).
